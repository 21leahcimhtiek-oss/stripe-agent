# Stripe Agent

> AI-powered conversational Stripe management platform

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/21leahcimhtiek-oss/stripe-agent)

## Features

- 🤖 AI-powered core features via OpenRouter
- 🔐 Authentication with Supabase
- 💳 Payments with Stripe
- 📱 Next.js 14 App Router
- 🎨 Modern UI with Tailwind CSS

## Quick Start

```bash
git clone https://github.com/21leahcimhtiek-oss/stripe-agent
cd stripe-agent
cp .env.example .env.local
npm install
npm run dev
```

## Environment Variables

See `.env.example` for all required environment variables.

## Tech Stack

- Next.js 14 (App Router)
- Supabase (auth + database)
- Stripe (payments)
- OpenRouter (AI)
- Tailwind CSS
- TypeScript
